# ERP-NSV-DEV

ERP-NSV-DEV is a development repository for NSV's internal business management system
based on ERPNext and Frappe, running on Docker infrastructure.

This repository is used for:
- ERPNext development environment
- HR module customization (Timesheet, Clock In/Out, Attendance, Reminder)
- Infrastructure and Docker-based deployment for NSV

> This repository is internal and intended for NSV development purposes only.
